# .dotfiles
ryul99's dotfiles ⚙️

## Overview
The `home/` directory mimics the actual home directory structure. Symlink mappings are defined in `file_mappings.json`.

## Installation
Run either command to create symlinks based on the mappings:
```bash
python3 install.py
dotfiles update
```

`dotfiles update` also fetches the latest updates from GitHub.
